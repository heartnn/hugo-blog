一、网上搜出了git为项目设置代理的方法：

git config --local http.proxy 'socks5://127.0.0.1:1080'
git config --local https.proxy 'socks5://127.0.0.1:1080'

此方法在我的电脑里并不适用，不过多讨论。

Ghost变得越来越成熟了，已跟随更新到0.11.3
