WordPress插件: External Links New Tab

2010-11-05 #wordpress

这款插件的作用是为所有的站外链接增加rel="external"，并且利用javascript将rel="external"转换为target="_blank"执行，目的是为了符合strict标准。

这款插件是由LDB External Links和JAB External Link New Tab修改而来，在wordpress 3.0.1下测试通过。

插件使用方法: 下载zip文件并解压缩，上传到/plugins ，然后在管理后台启用即可。
